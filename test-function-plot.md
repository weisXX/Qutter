# 函数绘制测试

以下是一个函数绘制示例：

```function
x^2
```

另一个示例：

```function
{
  "expression": "sin(x)",
  "xMin": -10,
  "xMax": 10,
  "yMin": -2,
  "yMax": 2,
  "title": "正弦函数图像"
}
```

还可以这样写：

```plot
{
  "expression": "cos(x)",
  "xMin": -2*3.14159,
  "xMax": 2*3.14159,
  "yMin": -1.5,
  "yMax": 1.5,
  "title": "余弦函数图像",
  "color": "#ff6b6b"
}
```
